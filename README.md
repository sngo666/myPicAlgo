# 基于qt和opencv开发的windows图像处理算法程序

程序中的算法并未调用opencv库，cv库仅用于数据转化和存储，目前已经实现了DFT变换和逆变换的内容，未来会更新更多的内容。
程序目前能够实现的功能不是很多，以后会慢慢完善。
这是本人为了了解和练习相关算法的内容所写的程序，供大家学习和参考。

想要build项目，你需要：

* mingw32
* opencv(4.5.1)(其他版本的兼容性未知)
* qt5
* qmake

> build方式  
> 执行start.bat, 或阅读脚本内容，单独执行你需要的指令。
